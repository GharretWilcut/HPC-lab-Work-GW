# How to use the get_benchmarks

### running comparison
```bash
chmod +x ./test_run.sh 
./test_run.sh ./data/Graph_1.txt 
```

### run full benchmark 
```bash
chmod +x ./run_benchmarks.sh 
./run_benchmarks.sh
```

### Generate data - ./data/generate_data.sh
 
```bash
chmod +x ./data/generate_data.sh 
./data/generate_data.sh
```
